background

faz com que você defina a cor de fundo.

background-image: url("imagem")

faz com que você defina uma imagem de fundo.

background-repeat: no-repeat / repeat / repeat-x / repeat-y.

não repete a imagem de fundo / repete a imagem de fundo / repete no eixo em direção a linha / repete no eixo em direção a coluna.

posiciona a imagem conforme os valores em porcentagem.

background-attachment: fixed / scroll

background-clip: border-box / padding-box / content-box;

o background ira até a borda / o background irá ate o padding / o background irá até o conteúdo.

background-size: contain / cover / numpx;

o tamanho da imagem de fundo ficará ele todo, sem cortes / ele ficará ele todo com cortes / baseado em seu valor em px.

text

deixa a última linha do texto a posição á direita / ao centro / a esquerda.

unicode-bidi: normal / bidi-override;

deixa o texto na ordem certa / deixa o texto na ordem contraria.

direction: rtl / ltr;

text-decoration-line: none / underline / overline / line-through;
